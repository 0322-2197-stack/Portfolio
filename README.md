# React + Tailwind Portfolio

Personal portfolio website built with React and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Customize

- Update hero text and bio in `src/App.jsx`
- Replace project cards in `src/App.jsx`
- Update contact links and email in `src/App.jsx`
